# COMP127 Class Demo Files

## Introduction

This repository contains a copy of demos performed in lecture for COMP127 Object-Oriented Programming and Abstraction course taught at Macalester College in Fall/Spring 20XX.


## How to Navigate

The demos for each class are placed inside the `src` folder in a separate directory prefixed with date of class in `MMDD` format, eg, `January 25th` class will have a directory that start with `0125`.
